# data/

This directory holds all raw inputs, generated datasets, and the live DuckDB database.

---

## Files

### `ufc-master.csv` ← included in repo
Historical UFC fight dataset — **6,528 fights, 118 columns**.

| Column group | Examples |
|---|---|
| Fighters | `RedFighter`, `BlueFighter` |
| Outcome | `Winner`, `Finish`, `FinishDetails`, `FinishRound` |
| Moneyline odds | `RedOdds`, `BlueOdds`, `RedDecOdds`, `BlueDecOdds`, `RKOOdds`, `BKOOdds` |
| Fighter stats | `RedAvgSigStrLanded`, `BlueAvgTDPct`, `RedWinsByKO`, … (per-corner) |
| Pre-computed diffs | `HeightDif`, `ReachDif`, `SigStrDif`, `WinDif`, … (Red − Blue) |
| Rankings | `RMatchWCRank`, `BMatchWCRank`, `BetterRank` |
| Context | `TitleBout`, `WeightClass`, `NumberOfRounds`, `EmptyArena` |

Source: public Kaggle dataset covering 2010–2024.
Load into DuckDB: `python -m edgelab_odds.ingest.loader`

---

### `edgelab.duckdb` ← generated
The live DuckDB database. Created automatically on first run.

```bash
# Inspect with DuckDB CLI
duckdb data/edgelab.duckdb
> SELECT count(*) FROM fights;
> SELECT * FROM features LIMIT 5;
```

Tables: `fights`, `features`, `model_runs`, `predictions`, `backtest_results`

---

### `fighters_live.csv` ← generated by `--all` scrape
Full UFC roster scraped from UFCStats.com. One row per fighter.
Columns: name, wins, losses, height_cms, reach_cms, stance, age,
avg_sig_str_landed, avg_sig_str_pct, avg_td_landed, avg_td_pct,
avg_sub_att, wins_ko, wins_sub, wins_dec_*, recent_form_pct, scraped_at

Regenerate:
```bash
python -m edgelab_odds.ingest.scraper --all
```

---

### `fighters_card.csv` ← generated by refresh
Stats for fighters on the next scheduled card only. Faster than a full roster scrape.

Regenerate:
```bash
python -m edgelab_odds.ingest.scraper --upcoming
# or as part of full refresh:
python -m edgelab_odds.ingest.refresh
```

---

### `odds_current.csv` ← generated by odds fetch
Latest moneyline odds from the-odds-api.com, normalised and vig-adjusted.

Columns: `fighter_a`, `fighter_b`, `odds_a`, `odds_b`, `imp_prob_a`,
`imp_prob_b`, `no_vig_a`, `no_vig_b`, `bookmaker`, `vig`, `commence_time`, `fetched_at`

Regenerate:
```bash
python -m edgelab_odds.odds.odds_api --fetch
```

---

### `odds_snapshots/` ← generated
Timestamped JSON dumps of every odds fetch. Used for closing-line value (CLV) analysis.

```
odds_snapshots/
├── odds_20250315T120000Z.json
├── odds_20250315T160000Z.json
└── ...
```

List all snapshots:
```bash
python -m edgelab_odds.odds.odds_api --history
```

---

### `cache/` ← auto-managed HTTP cache
Raw HTML from UFCStats and Tapology, cached for 6 hours per URL.
Safe to delete if you want to force a full re-scrape.

```bash
rm -rf data/cache/
```

## .gitignore notes

The following are excluded from version control (generated/large files):
- `edgelab.duckdb` — binary DB file
- `fighters_live.csv` — regenerate with scraper
- `odds_current.csv`, `odds_snapshots/` — live data
- `cache/` — HTTP cache
- `fighters_card.csv` — card-specific scrape

Only `ufc-master.csv` and this `README.md` are committed.
